Справка к программе оптимизации

О программе

Программа предназначена для решения задач оптимизации с использованием методов полного перебора и комплексного метода Бокса. Она позволяет находить оптимальные значения параметров для заданной целевой функции с учётом ограничений.

Теоретические сведения

1. Метод полного перебора

Метод полного перебора — это простейший метод оптимизации, который заключается в переборе всех возможных значений параметров в заданных границах с определённым шагом. Для каждой точки вычисляется значение целевой функции, и выбирается точка с наилучшим значением (минимумом или максимумом в зависимости от задачи).

Преимущества метода:

  • Простота реализации.
  • Гарантированное нахождение глобального оптимума при достаточно малом шаге.

Недостатки метода:

  • Высокая вычислительная сложность при большом количестве параметров или малом шаге.

2. Комплексный метод Бокса

Комплексный метод Бокса — это итеративный метод оптимизации, который использует набор точек (комплекс) для поиска оптимального решения. На каждой итерации худшая точка комплекса заменяется на новую, которая отражается относительно центра масс остальных точек.

Преимущества метода:

  • Более высокая скорость сходимости по сравнению с методом полного перебора.
  • Возможность работы с ограничениями.

Недостатки метода:

  • Может застревать в локальных оптимумах.
  • Требует настройки параметров (размер комплекса, коэффициент отражения).

Инструкция по использованию

1. Запуск программы

Для запуска программы выполните файл [название файла].exe. Программа автоматически создаст папку logs для хранения логов.

2. Выбор метода оптимизации

Программа поддерживает два метода оптимизации:

  • Метод полного перебора — перебирает все возможные точки в заданных границах.
  • Комплексный метод Бокса — использует итеративный подход для поиска оптимального решения.

3. Настройка параметров

Перед запуском оптимизации необходимо задать параметры:

  • Выбрать метод оптимизации.
  • Выбрать функцию для оптимизации.
  • Задать аргументы функции.
  • Задать ограничения первого рода (диапазоны значений переменных).
  • Задать ограничения второго рода (дополнительные условия).
  • Задать параметры для методов (шаг для метода полного перебора, погрешность для метода Бокса).
  • Ограничение на количество итераций (опционально).

4. Экспорт и импорт данных

Программа поддерживает экспорт и импорт данных в форматах Excel и TXT:

  • Импорт данных:
    • Данные можно загрузить из файла Excel (.xlsx) или текстового файла (.txt).
    • Файл должен содержать значения параметров и ограничений в определённом формате (см. примеры в папке examples).
  • Экспорт данных:
    • Результаты оптимизации можно экспортировать в файл Excel (.xlsx) или текстовый файл (.txt).
    • Экспортируемые данные включают оптимальные значения параметров, значение целевой функции и информацию о выполненных итерациях.

5. Запуск оптимизации

После настройки параметров нажмите кнопку "Рассчитать". Результаты оптимизации будут отображены в интерфейсе и сохранены в лог-файл.
